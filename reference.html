2.2.6 Responsive Design

---Your website design should include responsive design implementation using concepts covered in this course that would accommodate desktop, tablet, and mobile designs, including positioning, grid, and flex layouts.
---The use of contextual selectors to control sub level menus is required, including a navicon image for mobile design.
---Images should also be responsive for different screen sizes and devices.
(As you can see it in the source code and website:
---The website demonstrates a simple header with a navigation menu using HTML5 semantics.
---CSS styles define the layout and behavior of the navigation menu, including dropdown sub-level menus.
---Media queries are used to adjust the layout for screens smaller than 768px (commonly used for mobile devices),and for other smaller screens also making the navigation menu responsive by changing its display and layout.
---The responsive image styling ensured images adjust their size according to the screen width, maintaining their aspect ratio across different devices. )
